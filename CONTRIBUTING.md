# Contributing to PaletteOS

We love contributions! Please read these guidelines to make the process smooth and productive.

## Code of Conduct
Please follow standard respectful open source communication patterns when interacting with other contributors.

## How to Help
- **Bug Fixes**: Submit a PR targeting bugs or logic discrepancies inside our color conversion files.
- **Engine Improvements**: Optimizations to contrast calculation engines or accessibility checkers.
- **Feature Additions**: Propose features through Issues first before sending code modifications.

## Development Workflow
1. Fork and clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run tests using Vitest:
   ```bash
   npm run test
   ```
4. Confirm build compiling before committing:
   ```bash
   npm run build
   ```
5. Open a Pull Request targeting the `main` branch.

Thank you for making PaletteOS the best open-source color intelligence system!
